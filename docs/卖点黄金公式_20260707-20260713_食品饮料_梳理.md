## 关键共现关系

高表现行里最常出现的组件搭配：

- `成分配方原料 + 功效功能`：技术原因必须转成可感知结果。
- `成分配方原料 + 市场地位和权威背书`：用认证、检测、排名、专家/机构降低信任成本。
- `成分配方原料 + 痛点`：先指出问题，再给出原料解决路径。
- `成分配方原料 + 人群`：同一种成分需要针对宝妈、银发、白领等重写利益点。
- `成分配方原料 + 使用场景`：把抽象营养放进饭后、睡前、加班、健身、长辈照护等场景。
- `功效功能 + 品牌调性和故事`：老字号、国产、历史、古方等故事要服务于功效可信度。

## 可沉淀的内容语法

1. **身份/人群开头**
   - 适用：中老年、宝妈、儿童营养、女性、上班族等明确人群。
   - 结构：身份标签/专家或角色 -> 具体痛点 -> 成分/背书 -> 适用人群再确认。
   - 风险：只喊人群不讲问题，会变成泛泛广告。

2. **痛点-成分-权威链路**
   - 适用：保健食品、营养补充、睡眠、关节、补钙、免疫等。
   - 结构：身体/生活不适 -> 成分解释 -> 蓝帽/检测/认证/真人实测 -> 场景 CTA。
   - 风险：功效表达要注意合规，不能从“营养补充”滑到医疗承诺。

3. **风味-体感-场景链路**
   - 适用：饮料、零食、地方口味、轻健康食品。
   - 结构：味型/口感词 -> 入口动作或 texture 证据 -> 解馋/解腻/早餐/夜宵/饭后场景。
   - 风险：如果视频没有咀嚼、倒饮、拉丝、酥脆声等感官证据，风味词会空。

4. **权威-故事-人群链路**
   - 适用：老品牌、国产、蓝帽认证、儿童/长辈产品。
   - 结构：认证/品牌历史 -> 为什么适合目标人群 -> 功效/体感 -> 购买理由。
   - 风险：权威背书太早但没有痛点，会让 CTR 低。

5. **运输-原产地-新鲜安心链路**
   - 适用：地方特产、生鲜、节令食品、需要快递信任的商品。
   - 结构：产地/新鲜来源 -> 顺丰/包邮/冷链/包装 -> 到手体验 -> 适合谁买。
   - 风险：物流只是减阻，不是最终利益；仍要补“为什么值得吃/买”。

6. **产品代名词记忆链路**
   - 适用：有显眼包装、颜色瓶、昵称的产品。
   - 结构：昵称/包装识别 -> 核心成分或口感 -> 使用场景 -> CTA 复述昵称。
   - 风险：昵称如果没有前置解释，用户记住了名字但不知道买它干什么。

## 对 skill 的更新方向

已将 `短视频框架拆解1.0` 扩展为支持“卖点公式表 + 参考视频”的批量学习：

- 新增公式表读取流程：识别公式、关键词、消耗、CTR、人群和参考链接。
- 新增指标解释边界：高消耗、高 CTR 只能作为线索，不直接写成因果。
- 新增公式到内容功能的映射：把痛点、身份、成分、功效、权威、场景、包装、运输等转成视频里的创意职责。
- 新增参考视频抽样方法：优先高消耗、高 CTR、双高、对照组和人群覆盖。
- 新增视频不可访问规则：登录墙/403/无视频源时明确标注，不虚构内容。
- 新增食品饮料/健康食品内容理解框架：味觉、成分信任、痛点人群、权威合规、场景、原产地、包装运输、产品代名词、感官证据缺口。
